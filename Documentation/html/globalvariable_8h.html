Detailed Description

Global variables for saving status of all inputs.

Application layer and middle layer. Includes declaration of inputs of all appliances. Used for transferring data from input.h and bluetooth.h to controller. Also required for fan.h and led.h since it uses MAX_LED_BRIGHTNESS & MAX_FAN_SPEED.
